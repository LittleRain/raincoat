# Skills Roadmap

## Phase 1

- establish incubator structure
- create a basic template
- start incubating the first two skills

## Phase 2

- add a helper script for generating new skills from the template
- add a lightweight validation script for required files
- improve the skill catalog in `skills/README.md`

## Phase 3

- support exporting a skill into a standalone repository
- add optional release scaffolding for public open source publication
- add per-skill tests or CI only where the maintenance cost is justified
