# 交易业务周报生成器 v2.0

## 📁 工具文件

```
generate_report.py   ← 主脚本
app.js               ← 前端渲染模板（不要改动）
头部UPlist.xlsx       ← UP主列表（一般不更新）
README.md            ← 本文档
```

## 🚀 每周使用方法

### 1. 准备数据文件

把以下文件放到同一个文件夹：

| 文件 | 文件名包含关键词 | 用途 | 必需 |
|------|-----------------|------|------|
| 整体数据 | `整体数据` | S1核心趋势 | ✅ |
| 区分行业 | `区分行业` | S2行业+S3商家 | ✅ |
| 分行业×流量渠道 | `分行业` + `流量` | S4流量渠道 | ✅ |
| 分行业×内容类型 | `分行业` + `内容` | S5成交体裁 | ✅ |
| 商品×流量×商家… | `商品` + `流量` + `商家` | S4爆品归因 | 可选 |
| 商品×体裁×商家… | `商品` + `体裁` + `商家` | S5爆品归因 | 可选 |
| 头部UPlist | `UPlist` | 加林行业识别 | 首次需要 |

> 💡 **加速**: 大xlsx先另存为CSV，脚本自动优先读CSV

### 2. 运行

```bash
# 最简用法
python generate_report.py --dir 数据文件夹/

# 指定输出
python generate_report.py --dir data/ -o 本周周报.html
```

### 3. 打开

双击生成的HTML文件，浏览器打开即可。

## ❓ FAQ

**Q: 周编号怎么对应日期？**
A: 脚本自动取数据中最近4周的周编号，如W9/W10/W11/W12。周五到周四。

**Q: 缺少某个文件怎么办？**
A: 脚本会跳过缺失文件，对应模块显示空数据，不会崩溃。

**Q: 新增行业/类目？**
A: 修改脚本中的 `INDUSTRIES`/`CAT_MAP` 和 `classify_vec()` 函数。

**Q: UP主列表更新了？**
A: 替换 `头部UPlist.xlsx` 即可。
