# 标准化 Spec 摘要

## 周报目标

- 业务域：交易业务
- 读者：内部交易 owner、运营、分析同学
- 成功标准：用有数据依据的栏目级结论解释 GMV 和关键效率指标的周变化

## 时间定义

- 粒度：按周
- 本周：周五到周四
- 对比周期：上一完整周五到周四

## 栏目

1. 核心数据趋势
2. by 行业拆解
3. by 小店商家拆解
4. by 流量渠道
5. by 成交体裁

## 核心口径

- 平台业务线包含小店和自营
- 行业分类包含 ACG-南征、ACG-Allen、非ACG-悟饭、非ACG-加林、其他
- 关键指标包括 GMV、订单数、买家数、支付转化率、GPM
- 行业和商家重点观测阈值为 5%

## 输出合同

- 格式：HTML
- 栏目顺序：固定
- 必需区块：图表、表格、结论、数据说明
