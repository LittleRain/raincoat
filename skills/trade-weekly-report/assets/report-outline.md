# 周报结构

Populate each report section from the embedded contract in this skill.

Required structure:

1. report header
2. period metadata
3. section: 核心数据趋势
4. section: by 行业拆解
5. section: by 小店商家拆解
6. section: by 流量渠道
7. section: by 成交体裁
8. source-data notes

Section expectations:

- 核心数据趋势：
  - 小店与自营指标卡
  - GMV / 控比 / 买家 / 订单趋势图
  - 周波动结论
- by 行业拆解：
  - 整体行业图
  - 小店行业图
  - 整体 / 小店 / 自营三张表
  - 行业归因结论
  - `非ACG-加林 / 头部UP` 按文档10头部UP名单中的店铺ID识别
- by 小店商家拆解：
  - Top20 商家趋势图
  - Top20 表和分行业商家表
  - 商家归因结论
- by 流量渠道：
  - 总流量图
  - 小店 PV 占比图
  - 流量总览表
  - 分渠道表
  - 天马和 feed 专项
- by 成交体裁：
  - 整体 / 小店 / 自营饼图
  - 小店分行业饼图
  - 体裁对比表
