# Trade Weekly Report

这是一个面向交易业务周报场景的执行型 skill，用于读取指定 Excel/CSV 数据并生成交互式 HTML 周报。

这个 skill 会内置行业分类口径、栏目结构和分析规则，不依赖仓库外部文档才能理解或运行。

## 它做什么

这个 skill 会：

- 读取同一输入目录中的整体、行业、流量、体裁和可选明细文件
- 按“周五到周四”周口径生成交易业务周报
- 输出核心数据趋势、行业拆解、小店商家拆解、流量渠道、成交体裁五个栏目
- 生成可直接打开的 `report.html`

## 输入要求

把以下文件放入同一个输入目录。文件可以是 `.csv`、`.xlsx` 或 `.xls`，文件名允许带日期或周次后缀，但应满足以下 pattern。

## 数据口径要求

- 至少覆盖两个完整对比周期，否则无法生成“本周 vs 上周”分析
- 周期定义为周五到周四
- `业务线二级` 需要能区分 `小店` 和 `自营`
- 若列名存在尾部空格或轻微别名，脚本会做有限兼容，但不要依赖过度模糊的文件格式

## 输出内容

运行成功后会生成：

- `report.html`
  - 交互式周报页面
  - 包含图表、数据表、栏目级结论
- `run.log`
  - 执行日志
  - 当前 / 上周周期
  - 输入输出路径

## 如何运行

```bash
bash skills/trade-weekly-report/scripts/run-report.sh \
  --input-dir /path/to/input \
  --output-dir /path/to/output \
  --run-id trade-weekly-2026w14
```

## 最小使用步骤

1. 准备好 4 份必需输入文件
2. 确认字段齐全，且至少覆盖两个完整周期
3. 把文件放到同一个输入目录
4. 运行 `scripts/run-report.sh`
5. 打开输出目录中的 `report.html`
