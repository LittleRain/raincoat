# 周报结构

Populate each report section from the embedded contract in this skill.

Required structure:

1. report header
2. period metadata
3. section: 核心数据趋势
4. section: by 行业拆解
5. source-data notes

Section expectations:

- 核心数据趋势：
  - GMV 周趋势柱状图
  - 买家数周趋势柱状图
  - 下单转化率周趋势折线图
  - GPM 周趋势折线图
  - 周波动结论
- by 行业拆解：
  - 绘画+绘画周边 行业分析
  - VUP周边 行业分析
  - 两张行业内内容渠道分布表
  - 表格字段固定为：GMV、买家数、下单转化，且每个指标都展示上周环比
  - 商品归因结论
  - 指标波动与渠道占比结论
