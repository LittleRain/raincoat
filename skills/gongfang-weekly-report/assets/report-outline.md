# 周报结构

Populate each report section from the embedded contract in this skill.

Required structure:

1. report header
2. period metadata
3. section: 核心数据趋势
4. section: GMV 拆解
5. section: 流量 拆解
6. source-data notes

Section expectations:

- 核心数据趋势：
  - GMV 周趋势柱状图（坐标轴从0开始）
  - 买家数 周趋势柱状图（坐标轴从0开始）
  - 下单转化率 周趋势折线图（坐标轴从0开始）
  - 商品曝光PV 周趋势折线图（坐标轴从0开始）
  - 本周结论（GMV、买家数、支付订单数、下单转化率）
  - 波动超过5%的指标结论
- GMV 拆解：
  - 行业汇总摘要
  - 绘画+绘画周边 行业分析：
    - 指标卡片（GMV、买家数、下单转化率、商品曝光PV）
    - 4个折线图（GMV/买家数/转化率/曝光PV 过去4周，坐标轴从0开始）
    - 表格1：重点商家 TOP5（店铺名称、GMV含WoW、GMV占行业占比、买家数含WoW、下单转化率含WoW、商品曝光PV含WoW）
    - 表格2：重点商品 TOP10（商品名称、GMV含WoW、买家数含WoW、下单转化率含WoW、商品曝光PV含WoW）
    - 指标波动结论（仅展示>5%波动的指标）
    - 渠道占比变化结论（仅展示>5%变化的渠道）
    - 商品归因结论
  - VUP周边 行业分析：
    - 同上结构
- 流量 拆解：
  - 绘画+绘画周边 流量分析：
    - 内容渠道 GMV+曝光PV 双轴折线图（4渠道各一张，过去4周，坐标轴从0开始）
    - 内容渠道详情表格（GMV 4周、GMV占整体占比本周、商品曝光PV 4周、曝光PV占整体占比本周、PVCTR 4周）
    - 资源位二级入口 GMV+曝光PV 双轴折线图（过去4周，坐标轴从0开始）
    - 资源位二级入口详情表格（GMV 4周、GMV占整体占比本周、商品曝光PV 4周、曝光PV占整体占比本周、PVCTR 4周）
    - 流量归因结论
  - VUP周边 流量分析：
    - 同上结构

关键约定：
- 每个表格的每个指标格须同时展示"本周数值"和"上周环比"，格式为"数值 (环比%)"
- 仅解释指标环比波动超过5%的行业指标
- 仅解释占比变化超过5%的内容渠道
- 所有图表坐标轴从0开始，禁止截断
