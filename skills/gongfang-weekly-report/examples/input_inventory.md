# 输入文件清单

将以下文件放入同一个输入目录，文件名允许带日期或周次后缀，但应满足对应 pattern：

- `整体数据*`
  - 字段：日期、PVCTR、商详UV、支付订单数、支付订单买家数、GMV（不减退款）、商详支付转化率-UV、GPM
- `行业数据*`
  - 字段：日期、后台一级类目名称、商品曝光PV、PVCTR、商详UV、支付订单数、支付订单买家数、GMV（不减退款）、商详支付转化率-UV、GPM
- `行业商品明细数据*`
  - 字段：周、商品名称、后台一级类目名称、商品曝光PV、PVCTR、商详UV、支付订单数、支付订单买家数、GMV（不减退款）、商详支付转化率-UV、GPM
- `内容渠道数据*`
  - 字段：日期、后台一级类目名称、内容类型、商品曝光PV、PVCTR、商详UV、支付订单数、支付订单买家数、GMV（不减退款）、商详支付转化率-UV、GPM
