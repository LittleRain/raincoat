数据说明
1. 数据来源：整体数据、行业数据、行业商品明细数据、内容渠道数据、商家销售明细、资源位二级入口数据
2. 周期定义：每周五到周四为一个完整周
3. GMV 口径：不减退款
4. 下单转化率 = 支付订单数 / 商详UV
5. PVCTR = 商详UV / 商品曝光PV
6. GPM = GMV / 商品曝光PV × 1000
7. 结论仅解释环比波动超过5%的指标
8. 绘画与 VUP周边均为独立分析口径
