# 标准化 Spec 摘要

## 报告目标

- 业务域：圈子内容（日报）
- 读者：圈子业务运营同学、分析同学
- 目标：输出总览 + 圈子内容 + 漫展双路径 + 项目转化的日报结论

## 时间定义

- 粒度：daily
- 统计周期：昨日（Asia/Shanghai）
- 对比周期：无固定环比/同比，按当日排序和场景差异归纳

## 固定栏目

1. 总览：按圈子/路径展示曝光、点击、CTR、互动率
2. 绘画圈：内容分析（Top10，按曝光排序）
3. 模型圈：内容分析（Top10，按曝光排序）
4. 漫展圈：T3 vs 票务商详内容分析（双路径对比）
5. 漫展圈：项目转化分析（Top 项目与漏斗承接）

## 数据合同（最小集）

- 绘画圈 feed：`*拉取绘画圈feed流明细数据*`
- 模型圈 feed：`*拉取模型圈feed流明细数据*`
- 漫展圈 T3 feed：`*拉取漫展圈feed流明细数据*`（排除“票务商详”）
- 漫展转化：`*拉取漫展项目的转化行为数据*`
- 漫展票务 feed：`*拉取漫展圈feed流明细数据_票务商详*`

## 关键口径

- 绘画圈/模型圈/漫展圈 T3：PV 口径（曝光pv、点击pv、pv_ctr、pv互动数）
- 漫展票务 feed：UV 口径（uv_expose、uv_click、uv_ctr）
- 漫展项目转化：想去数、商详浏览uv、讨论区曝光uv、讨论区点击uv、uv_ctr

## 分析边界

- 允许：按曝光排序、特征归纳、路径差异对比、基于字段证据的多角度结论
- 禁止：无数据支撑的归因、跨口径混算、缺文件时伪造结论

## 输出合同

- 输出格式：HTML
- 区块顺序：标题区 -> 5 个栏目 -> 数据说明区
- 每个栏目至少包含：表格/对比视图 + 带指标证据的栏目级结论
