# 输入文件清单样例

- `*拉取绘画圈feed流明细数据*`
  - 绘画圈昨日 feed 流内容浏览数据，by 内容
- `*拉取模型圈feed流明细数据*`
  - 模型圈昨日 feed 流内容浏览数据，by 内容
- `*拉取漫展圈feed流明细数据*`（不含"票务商详"）
  - 漫展圈 T3 入口昨日 feed 流浏览数据，by 内容
- `*拉取漫展项目的转化行为数据*`
  - 票务商详→漫展圈的昨日转化数据，by 项目
- `*拉取漫展圈feed流明细数据_票务商详*`
  - 漫展圈票务商详入口昨日 feed 流浏览数据，by 内容（UV 口径）

脚本会优先读取 `.csv`，其次读取 `.xlsx/.xls`。
