# HTML 合同

生成出的日报 HTML 必须按以下顺序包含区块：

1. 标题区（含报告日期）
2. 总览：圈子整体表现
3. 绘画圈：内容分析
4. 模型圈：内容分析
5. 漫展圈：T3 vs 票务商详内容分析
6. 漫展圈：项目转化分析
7. 数据说明区

总览区必须包含：

- 按圈子/路径的汇总表（曝光、点击、CTR、互动率）
- 至少 2 条场景级对比结论

绘画圈/模型圈内容分析区必须包含：

- Top10 内容表（按曝光排序）
- 字段：标题、副标题、发布时间、曝光、点击、CTR、互动数、链接
- 至少 3 条带指标证据的结论

漫展双路径内容分析区必须包含：

- T3 路径 Top10 表（PV 口径）
- 票务商详路径 Top10 表（UV 口径）
- 路径差异结论（CTR、类型/分区、集中度、重合项目）

漫展项目转化分析区必须包含：

- 项目 Top10 表（按商详浏览 UV）
- 曝光承接率、点击承接率、UV CTR
- 至少 2 个代表性项目的原因分析
